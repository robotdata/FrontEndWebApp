What does this tool do for you?

Below is a simple demonstration for the tool. As you see, one can overlay a 3D cad model on the 2d image by simply clicking on the buttons as shown below.

The output image can be further used to train neural networks as well.

Applications

  1. Object Proposal Generation: In object proposal generation, one estimates the likely bounding boxes that are likely to be objects. The ObjectNet3D gives the bounding box coordinates in its output, making sure that one can use this tool for object proposal generation.
  2. 2D Object Detection: In object proposal generation, one estimates the places which are likely to be objects. However, since the output also contains what object does it exactly is, one can train the neural network based on these labels and further do 2D object detection.
  3. Continuous 3D Pose Estimation: The output of this tool contains the azimuth angle, the elevation angle and the inplane rotation of the CAD Model. This gives the 3D pose estimation of the object with respect to the CAD model.

Output File:

  1. Filename: This field simply gives the name of the file you annotated.
  2. Objects: The objects field is a list of multiple objects.
  3. class:This field determines the class of the object.
  4. bbox:This field gives the top left and the bottom right coordinates of the bounding box.
  5. Azimuth Coarse:This field gives the azimuth angle of the CAD Model.
  6. Elevation Coarse: This field gives the elevation of the CAD Model
  7. Distance: This is the distance from the observer to the center of the object.
  8. Theta: This is the inplane rotation of the CAD Model that os used to get the 3d pose of the object.
  9. Truncated: This field tells whether the object is truncated or not. It is 1 if the object is truncated else it is 0.
  10. Occluded: This field is set to 1 if the object is occluded.
  11. Difficult: This field is set to 1 if the object is difficult to annotate.

Major Advantage: Simplicity of use

We have packaged the application and now we do not need Matlab to run the application. One can simply install the application and then run it as a desktop tool.

Below are the instructions on how to download and install the tool.

Downloading and installing

  1. Download the tool from here
  2. Click on the application and you will see the window as follows:

  3. Add your installation path

After Installation

  1. Once installed, open the application and you would something as below:

  2. Download the root directory from here and copy paste it in any location in your computer
  3. The root directory will look like this:

New Images

  1. Add images and do the corresponding annotation for that. To do that click on add images and get annotation.
  2. You will be asked to add image directory
  3. Go ahead and add your image directory
  4. Once done, add annotation directory where you wish to store the annotations. Add the number of objects for each of the images.
  5. For every object, enter the class of the object

New CAD models

  1. Click on add new Cad Models
  2. Add Root Obj Files
  3. Enter the folder where you want to store the new CAD file

Start Annotating

  1. Click on call annotate pose.m and add annotation, the image and the cad directory from the root

Store Output

  1. Click on Store Image annotations to store the output
  2. A dialog box will ask for the root directory
  3. Go ahead and add your directory
  4. A dialog box will ask for the output directory
  5. Go ahead and add your output directory
  6. Once the output directory has been added, go ahead and store your outputs
  7. You can view the outputs in the output directory
